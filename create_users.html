File passphrase (facoltativo)

Carica un file JSON con {username: passphrase} per popolare automaticamente le passphrase nella sezione rotazione PAT qui sotto. ⚠ Il file contiene password in chiaro: custodiscilo con attenzione (password manager, USB crittato, ecc.). Non committarlo mai.
Formato atteso: {"raffaele": "adminpass", "marco": "userpass", …}
Il file rimane solo in memoria del browser, non viene salvato automaticamente. Esportalo quando aggiungi nuovi utenti per aggiornare la tua copia sicura.

Rigenera token (rotazione PAT)

Quando un PAT scade o viene compromesso, devi sostituirlo in tutti i payload cifrati degli utenti. Per farlo servono le passphrase di ogni utente. Inserisci il nuovo PAT, poi le passphrase riga per riga.

Quale token vuoi ruotare?

token_data: ruota il PAT del repo CollinettaAI-data. Devi conoscere le passphrase di TUTTI gli utenti.
token_auth: ruota il PAT del repo CollinettaAI-auth. Devi conoscere le passphrase di tutti gli ADMIN.
Il nuovo token sostituirà quello attuale nel payload cifrato di ogni utente interessato.
